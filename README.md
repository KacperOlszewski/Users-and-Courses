Technologies: Angular2, Typescript, webpack, Jasmine, angular2-mdl
--------------


Requirements
==============
node >= 6.1.0


Installation
=============
npm install


Deploying DEV
=============
npm run serve -(will run and watch on 127.0.0.1:8085)


Deploying PROD
=============
npm run build:prod - (will build package in /dist directory)


Testing
=============
npm run test
